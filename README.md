# RageMines Custom Swords

Custom sword textures for RageMines server

## Installation

1. Copy the texture pack folder to your Minecraft resource packs directory:
   - Windows: `%appdata%\.minecraft\resourcepacks`
   - macOS: `~/Library/Application Support/minecraft/resourcepacks`
   - Linux: `~/.minecraft/resourcepacks`

2. In Minecraft, go to Options > Resource Packs
3. Select "RageMines Custom Swords" and move it to the right side
4. Click "Done"

## Usage

All custom swords are based on netherite swords with Custom Model Data.

Total custom swords: 84

## Give Commands

### Minecraft 1.20.5+ (New Format)
```
/give @p minecraft:netherite_sword[custom_model_data=X]
```

### Minecraft 1.20.4 and earlier (Legacy Format)
```
/give @p minecraft:netherite_sword{CustomModelData:X} 1
```
